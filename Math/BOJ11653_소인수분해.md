### 소인수분해
- - -
## 변수 설명
- N : input
## 함수 설명
- PF : Prime Factorization, 매개변수 n을 소인수분해하여 출력해주는 함수
## 코드 설명
- 바깥 for문 i는 2부터 i*i이 n보다 커질 때까지 1씩 증가한다.
- 안쪽 while문은 n * i가 0이 될 때까지 i를 출력해준 뒤 n /= i 해준다.
- for문이 끝나면 마지막으로 n이 1보다 크다면 출력해준다.
## 기억해야 할 것
- 소인수분해 공식을 기억하자.
