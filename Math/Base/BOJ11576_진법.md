Baekjoon 11576
=============
Base Conversion  <https://www.acmicpc.net/problem/11576>
---------------
- - -
###  진법 변환
- - -
## 변수 설명
- a,b : a진법에서 b진법으로 변환
- ans : a진법을 변환한 10진법
## 함수 설명
- convert : a진법을 10진법으로 변환한 변수(num)와 base(b진법)을 변환해주는 함수
## 코드 설명
- a를 10진법으로 바꿔준다. (ans 변수)
- convert 함수에 ans와 b를 매개변수로 넣어주고 실행시킨다.
## 기억해야 할 것
- a진법 => b진법 방법
- a진법 => 10진법 => b진법
