### 모음인지 아닌지 판별
- - -
## 변수 설명
- vowel : a,e,i,o,u 를 원소로 가지는 배열
- str1 : 문자열 (input)
- T : test_case
- flag : 모음이면 false, 초기값은 true
## 코드 설명
- str1을 입력받아서 모음인지 검사한다.
- 모음이면 flag를 false로 바꾸고 break시킨다.
- 내부 for문을 나와서 flag가 true이면 str[i]를 출력시켜준다.
- 반복문 마지막에 flag를 true로 바꿔준다.
## 기억해야 할 것
- 바로바로 출력시켜주는 코드 방식에 익숙해지자!
