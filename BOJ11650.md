좌표 정렬하기  <https://www.acmicpc.net/problem/11650>
---------------
- - -
###  C++ 함수 sort 사용하기
- - -
## 변수 설명
- N : 좌표 개수 (input)
- x, y : x, y 좌표 (input)
- v : pair<int, int>를 자료형으로 갖는 vector 
## 코드 설명
- N개 만큼 x,y를 입력받는다.
- x,y를 v에 넣어준다.
- v를 처음부터 끝까지 sort해준뒤 출력해준다.
## 기억해야 할 것
- sort 함수
