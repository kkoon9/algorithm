Baekjoon 10814
=============
나이순 정렬  <https://www.acmicpc.net/problem/10814>
---------------
- - -
### sort 매개변수에 함수를 넣어줄 수 있다.
- - -
## 변수 설명
- Person : string형 자료형인 name과 int형 자료형인 age, join를 갖는 구조체
- age : 사람의 나이 (input)
- join : 들어온 순서
- n : 사람 수 (input)
- a : Person 자료형을 갖는 벡터
- - -
## 함수 설명
- cmp : age를 비교해준 뒤 join을 비교해주는 함수
## 코드 설명
- n만큼 age와 name을 입력받는다.
- a를 cmp 기준으로 정렬해준 뒤 출력해준다.
## 기억해야 할 것
- 소문자 a의 아스키 코드는 97이다.
