### 동적 계획법(Dynamic Programming)을 사용하는 문제

### 가장 작은 값을 더해주며 최소값을 출력해준다.
- - -
### 변수 설명
- result1 : W 칠한 결과
- result2 : B 칠한 결과
- result3 : R 칠한 결과
- x : W, B, R을 입력 받기위한 변수
- W, B, R : 각 줄에 있는 W, B, R의 개수
