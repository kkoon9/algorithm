### modular 연산에 대해 공부하자.
- - -
## 변수 설명
- A, B, C : input
## 코드 설명
- modular 연산에 대해 정확히 알고 있는지 묻는 문제이다.
- (A+B) % C --- 1
- (A % C + B % C ) % C --- 2
- (AxB) % C --- 3
- (A % C x B % C ) % C --- 4
- 1과 2, 3과 4의 결과가 같다.
## 기억해야 할 것
- modular 연산의 규칙을 잘 기억하자.
