파도반 수열 <https://www.acmicpc.net/problem/9461>

### 다이나믹 프로그래밍을 사용하자.
- - -
## 변수 설명
- memo : 정삼각형의 변의 길이를 가지는 배열
- t : test_case
- n : input
## 코드 설명
- N이 8까지는 규칙을 찾기 어렵다.
- N이 9부터는 P(N) = P(N-1) + P(N-5) 이라는 규칙을 찾을 수 있다.
## 기억해야 할 것
- 모든 문제에는 규칙이 있다.
- 규칙을 찾지 못하겠으면 그림을 그려보자.
